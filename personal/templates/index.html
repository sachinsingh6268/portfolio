BASIC   BANKING   SYSTEM

As the name suggests, it's a basic banking software where you can keep record of your banking transactions.

Simply you have to enter the name of the receiver and his/her email address along with the sender's name and his/her basic credentials. Here sender name is being taken so that you can also keep records for your family members, friends, relatives. After entering the details and Amount to be paid please click *Pay Now* button. After successful of the payment your transaction history will be shown in a seperate page. This history will be as long as you want that. If you want to delete any specific transaction, you can delete it by clicking a button given at the right most corner of that particular transaction.

E-COMMERCE   WEBSITE

This is a E-commerce website. Customer can visit the product by clicking on the "Quick View" button on the product section. It has different sections for different types of product. Customer can add the product in the cart just by clicking on the button "Add To Cart". After choosing the products and putting those into the cart,Customer can go to the checkout page by clicking "Cart(n)" button on the top right corner of the website where he/she can preview the items selected and can fill the delivery and billing address. After placing the order, Customer can track his/her order where he/she will get the latest about order placed.
SmartCart is also provided with "Contact Us" page so that if Customer is facing any problem related to the product quality or any specific issue, they can reach out to us. We will get back to them as soon as possible.

TEXT   ANALYSER

It's simple program based on Python where any user can make a redundant part of a text useful by using certain operations that are provided in this program.

Till now, it has only some specific operations that can be applied such as Remove Punctuations, Capitalise Text, Remove unnecessary extra spaces, count the number of character, etc. I am working on it so that it can handle more operations on text and help anyone to make a provided text useful.

LIBRARY   MANAGEMENT   SYSTEM

It's a simple Python based program that can create Library and manages different tasks very easily that are very much used in Library. Not only it creates Library but also it can work on already existing libraries.
It keeps records of all libraries that the user has created for different uses. It can help in keeping the records of the books that are available for readers, person information who has taken the book from the library, the people record who have made contribution by donating books to make the library more robust, and it also updates the library as a reader returns the book back to the Library.

HEALTHY   PROGRAMMER

As we all know a most of the Programmers sit on a place and keeps on doing his/her Programming without caring of health. This unconsciousness by the Programmer is making life of programmer little difficult if we talk about their Physical and Mental health.
To get rid off these problems, I have written a simple program that will remind a programmer to do certain physical and mental exercises in certain time. I have taken 3 things to do by programmer as Drinking Water, Doing eye exercise, Doing physical exercise. Once the program is started it will play a music(if it's reminding of drinking water then it will play a sound of water) after a specified time. To confirm that if programmer has completed the exercise he/she can write what he want to write at that time (e.g. taken 500ml water). It will be saved in a file with the date and time so that programmer can analyse and try to improve his/her health during working hours.
